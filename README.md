# Flask_login

> Status: Desenvolvimento 


## Instalação

Use o gerenciador de pacotes [pip](https://pip.pypa.io/en/stable/) para instalar as dependências necessárias

##### Windows:
```zsh
pip install -r requirements.txt 
```

##### macOS/Linux:
```zsh
pip3 install -r requirements.txt
```

## Para rodar

##### Windows:
```zsh
python app.py
```
##### macOS/Linux:
```zsh
python3 app.py
```
